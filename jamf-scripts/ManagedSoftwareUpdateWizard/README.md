# msu-dialog.command

A swiftDialog workflow that uses apiutil-msu.zsh and API Utility to perform Managed Software Updates. To run, double-click on the script in Finder, or run the following command:

```
./msu-dialog.command
```

## apiutil-msu.zsh

A command line tool for creating and reporting on managed software updates.

For help:

```
./apiutil-mdm-commands.zsh --help
```
